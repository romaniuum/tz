Frequently asked questions

What is a dedicated server ?

A dedicated server refers to a computing machine that offers all of its resources without sharing with others. In contrast to a virtual server, it allocates its entire processing power, memory, and storage solely to a single user. This setup is frequently utilized in various scenarios, particularly in cloud computing, for "bare metal" services that ensure complete access to the physical resources of the server.

How does a dedicated server work ?

Each dedicated server is operated through a console that enables you to set up your applications, enhance security, and monitor performance in real-time. For comprehensive details, check our ScalewayDocs.

What distinguishes a dedicated server from shared hosting ?

Unlike shared hosting, which distributes a server’s resources among several clients, a dedicated server is entirely yours, ensuring superior performance and enhanced security.

Why choose a dedicated server ?

Dedicated servers are ideal for businesses that require substantial processing power, increased security, and personalized management, allowing precise adjustments to meet your technical requirements.

How to rent a dedicated server ?

Selecting and configuring your dedicated server at Scaleway is straightforward: choose the specifications that suit your needs while benefiting from our commitment to reduce environmental impact. For further details, please refer to our page on environmental responsibility.

How to set up a dedicated server ?

You can follow a step-by-step guide for installing your server available on ScalewayDocs, which provides comprehensive support to effectively establish and launch your server based on your criteria.

What is the difference between a dedicated server and a VPS ?

A VPS is a more affordable option that shares a server’s physical resources among many users, while a dedicated server offers exclusive resources for unmatched performance.

Use cases for dedicated servers

Whether for online gaming, e-commerce stores, or enterprise applications, dedicated servers provide the stability, capacity, and security necessary to manage your critical operations.

What are the differences between dedicated and shared servers?

A shared server is a solution where multiple clients use the same resources of a physical server. This sharing reduces costs but limits flexibility and performance. Conversely, a dedicated server is allocated to a single client. This means all the server's resources are exclusively available to that client, offering maximum power and flexibility.

Control: A dedicated server offers complete control, allowing the user to configure settings as desired. In contrast, a shared server allows limited customization due to the shared use of resources.

Performance: A dedicated server, by not sharing its resources, offers optimal performance. A shared server, on the other hand, may experience slowdowns if another site on the same server has high activity.

Security: Dedicated servers are more secure due to their isolation. Shared servers may present risks if one of the sites sharing the server is compromised.

Advantages of a dedicated server over a shared server

A dedicated server has undeniable advantages over a shared server. First and foremost, it offers optimal performance. All server resources (CPU, RAM, disk space) are entirely dedicated to your use, which helps optimize the speed and responsiveness of your applications.

Total control: You have the freedom to configure the server according to your specific needs. This includes the ability to install the latest generation processors, adjust the RAM, and manage storage space.

Enhanced security: As the sole user of the server, you are protected from security issues that may arise in a shared environment. You can also implement stricter data protection measures.

Exclusive resources: You do not have to share resources with other users, ensuring constant availability and stable performance, even during traffic peaks.

Customization: With a dedicated server, you can fully leverage virtualization technologies and have different options at your disposal.

Disadvantages of a dedicated server versus a shared server

Despite its numerous advantages, a dedicated server also has some disadvantages that should be highlighted.

Technical management: Owning a dedicated server requires technical skills for installation, administration, and maintenance. Unlike a shared server, the management of a dedicated server is entirely up to you.

Cost: A dedicated server is more expensive than a shared server. In addition to the purchase or rental cost, one must also consider the cost of energy, cooling, and maintenance.

Non-shareable resources: In a dedicated server, unused resources cannot be shared with other users, which can lead to resource wastage.

Limited flexibility: Unlike a shared server or VPS, the capacity of a dedicated server cannot be quickly and easily increased or decreased.

Backup: In the event of a failure, a dedicated server does not have an automatic backup system as is the case in a shared server environment, unless specific backup measures have been implemented.

Dedicated server or shared server - which to choose?

The choice between a dedicated server and a shared server largely depends on your performance, control, and budget requirements.

Performance: If your web application or site requires significant resources and needs to handle a high volume of traffic, a dedicated server is the best option. It guarantees optimal performance, unaffected by other users' activities.

Control: If you need the freedom to install and configure your own applications, a dedicated server offers complete control. You can adjust the server settings according to your specific needs.

Budget: If your budget is limited, a shared server can be a more economical solution. However, it is important to consider that performance and control are limited by the sharing of resources with other users.

What's the difference between a dedicated server and a virtual server (VPS)?

A virtual server, also known as a VPS (Virtual Private Server), is a virtual machine hosted on a physical server. Unlike dedicated servers, they share the hardware resources of this host server with other VPSs.

Control and customization: Both types of servers offer full control, allowing the installation of your own operating system and configuration settings. However, with a dedicated server, you have exclusive access to all hardware resources, which can provide superior performance.

Cost: Virtual servers are generally less expensive than dedicated servers, as they share the resources of a physical server.

Scalability: Virtual servers are more easily scalable, allowing for quick and easy adjustment of allocated resources.

Security: Due to physical isolation, a dedicated server offers a higher level of security. A VPS, although virtually isolated, shares the same hardware as other VPSs, which can pose additional security risks.

Bare-metal servers (cloud servers): an alternative to dedicated servers?

Bare-metal servers, also known as cloud servers, are an interesting alternative to dedicated servers. These servers, rented from a provider, are not shared with other tenants. This results in superior performance and high adaptability of resources.

Unlike traditional cloud computing based on virtual machines, bare-metal servers are not provided with a preinstalled hypervisor. This offers greater flexibility in terms of configuration and management.

Bare-metal servers can be an excellent option for small and medium-sized businesses seeking a high-performance and cost-effective hosting solution. The ability to quickly automate and adjust resource allocation can be a significant advantage for these businesses.

Dedicated private servers: what for?

A private dedicated server proves useful for a variety of uses. It is particularly valuable for projects that require a large amount of resources and complete isolation of those resources, ensuring optimal performance.
Advanced web hosting: High-traffic websites, large-scale online stores, or resource-intensive web applications can benefit from the control and performance offered by a private dedicated server.
Game hosting: Multiplayer online games often require their own servers for an optimal gaming experience. A private dedicated server offers the necessary customization freedom and capacity for these types of applications.
Storage and database infrastructures: Private dedicated servers are ideal for hosting large databases, CRM or ERP systems, or for serving as file or backup servers.
Application development and testing: An isolated environment provided by a private dedicated server is perfect for developing, testing, and deploying applications.
Virtualization: Due to their power and flexibility, private dedicated servers are often used as hosts for setting up virtual private servers (VPS).
It is important to note that choosing a private dedicated server should be based on the specific needs of each project. It is recommended to carefully evaluate the requirements in terms of performance, security, and control before making a choice.

Save money on your dedicated server: Scaleway tips

To optimize your expenses on a dedicated server, several tips can be implemented.
Resource optimization: Efficient use of your server’s resources can save money. Regularly assess your needs and adjust your server's resources accordingly.
Operating system choice: Choosing Linux as your operating system can save you the licensing fees associated with Windows.
Energy consumption: Modern dedicated servers are more energy-efficient. Consider an upgrade if your server is old.
Outsourcing certain tasks: Tasks like email sending can be outsourced to reduce the load on your server.
Use of automation solutions: These solutions can reduce the time spent managing your server, resulting in indirect savings.
Reducing unnecessary traffic: Using tools to block malicious bots can reduce unnecessary traffic on your server.

Free dedicated servers: myth or reality?

The idea of a free dedicated server may seem enticing, but is it really feasible? The truth is that even if some hosting providers offer free trials or free hosting periods, these offers are often time-limited or come with restrictions in terms of resources and features. These offers are not truly "free" as they often require a longer-term commitment or the subscription to additional services. Therefore, it is essential to carefully read the terms of these offers before committing.
